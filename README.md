A **production-grade full-stack platform** where users track and share real-world contributions to the **UN Sustainable Development Goals** with challenges, social features, gamification, and a **24-view analytics suite**.

**React 19 · Node.js · PostgreSQL · Google OAuth · 70+ REST endpoints · 24 analytics views**

Key Contributions:

• Designed the PostgreSQL schema from the ground up — **15 tables, 5 materialized views, and GIN full-text search indexes**  
• Engineered **70+ REST API endpoints** for authentication, social interactions, challenges, and analytics  
• Built **24 analytics dashboards** including cohort retention, churn prediction, geographic heat maps, and network graphs  
• Implemented **Google Identity Services OAuth** with server-side token verification  
• Secured the backend using **bcrypt hashing and fully parameterized SQL queries**

• Architected reusable hooks, utilities, and components across a **33-file analytics frontend**
